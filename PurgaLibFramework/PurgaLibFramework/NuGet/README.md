# PurgaLib Framework
PurgaLib is a lightweight framework for SCP: Secret Laboratory.

Features
- Event-driven architecture
- Modular design
- Developer-friendly API 

Status **RELEASE**

Usage
Add `PurgaLibFramework.dll` to LabAPI/plugins/Global and reference it in your plugin project.

For version 2.0.0 ChangeLogs:
- Fixed events on Harmony
- Everything now loads on it's own and doesn't need LabAPI (only for Loader to be called.)
